# API_endpoints

Using the Python requests library, extract data from the following DummyJSON endpoints:
 
Products endpoint
https://dummyjson.com/products 
 
Carts endpoint
https://dummyjson.com/carts 
 
For each endpoint:
- Send a request using Python.
- Extract the JSON response.
- Convert the data into a well-structured DataFrame.
- Save the final output as a CSV file.
